# 🚀 Image Booster V1 — Raw Rust & WebAssembly Power ONLY 1.77 MB

Yoi, welcome to the repo! I didn't just build another heavy, bloated electron app that litteraly just JUMPS all your RAM. I (and femini) went full-on high-on-VS_code mode and engineered a split-pipeline desktop beast. 

### 🛠️ The Big One:
* **The Frontend (`index.html`):** A custom, sleek dark-mode studio running canvas-level image engines that doesn't even let you drag and drop ):  but works perfecty=ly fine and lowkey mature enough to be a normal app. It gives crisp, satisfying 40% volume audio pop feedback on every click and includes zero-bug vector cropping handles.
* **The Brains (Rust + WASM):** Instead of slow JavaScript loops, your images are blasted directly into low-level **WebAssembly** compiled straight from blazing-fast Rust logic. It crunches pixels instantly at bare-metal speeds.
* **The Shell (Tauri):** We packaged everything natively using Tauri v2. No heavy Chromium engine hiding in them buhes. The entire desktop `.exe` installer is literally **1.77 MB**!
